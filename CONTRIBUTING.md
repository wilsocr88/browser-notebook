# CONTRIBUTING

Thanks for your interest in this project. Contributions are welcome.

Feel free to [submit a pull request](https://github.com/wilsocr88/browser-notebook/pulls)
or, if you find a bug, [an issue](https://github.com/wilsocr88/browser-notebook/issues)
(make sure it's a bug in the component, and not in your code or toolset!).
